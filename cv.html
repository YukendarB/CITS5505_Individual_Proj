Featured Project

DevQuest - Interactive Web Learning Platform

DevQuest is a multi-page interactive web application developed using HTML, CSS, JavaScript and Bootstrap. The project is designed to teach users core web development concepts through a structured tutorial page, a dynamic quiz page, an AI reflection log and a personalised CV page.

This project demonstrates a polished multi-page interface, interactive learning features, dynamic quiz logic and a consistent custom dark theme built for an engaging student learning experience.

  • Built a modular front-end web application using HTML, CSS and JavaScript with a reusable component-based design system.
  • Implemented client-side data handling using JSON and dynamic DOM rendering for quiz generation and interaction.
  • Designed event-driven UI interactions including real-time validation, progress tracking and conditional feedback.
  • Utilised browser APIs such as localStorage for persistent state management and fetch() for asynchronous data retrieval with error handling fallback.
  • Developed interactive learning tools including a DOM manipulation demo, CSS state switcher and embedded code execution playground.
  • Ensured responsiveness and accessibility through structured semantic HTML and adaptive layout design.
